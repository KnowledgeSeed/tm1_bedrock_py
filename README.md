# tm1_bedrock_py
A python package by Knowledgeseed for TM1 Bedrock.

## Requirements
* Python
* TM1py
* mdxpy
* Pandas

## Development

```
python -m venv .env
source .env/bin/activate
python -m pip install -r requirements.txt
python -m build
```

## Usage

Create a connection in TM1 with at least the following parameters set:

* address
* user
* password
* port
* ssl
